# hugo-theme-ling
基于[hyde](https://github.com/spf13/hyde) 修改的主题，增加了搜索、翻页等功能

## 截图
![theme](images/theme.png)

## 安装
```
$ cd your_site_repo/
$ mkdir themes
$ cd themes
$ git clone https://github.com/jangworn/hugo-theme-ling ling

```
## 启动
修改config.toml   
theme = "ling"
```
$ hugo server --buildDrafts
```
